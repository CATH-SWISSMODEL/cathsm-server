Protein Modelling Workflow

  • The CATH-SM workflow allows users to generate 3D models from protein sequences.
  • This work is a collaboration between CATH (protein structure classification) and SWISS-MODEL (protein structural modelling) as part of an ELIXIR Implementation Study.

Try It Learn more

About

This workflow uses CATH to search a protein sequence against Functional Families (FunFams) to try and identify high quality template structures. These template structures are then submitting to SWISSMODEL in order to generate the full 3D models.
